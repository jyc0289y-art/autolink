다음 작업은 **본 SOW에 포함되지 않으며, 별도 견적 대상**입니다:

- 의뢰자 회사 내부 시스템(ERP·그룹웨어·POS·회계 SW) 직접 연동
- 캡차·SMS 인증·생체 인증·OTP·공인인증서가 필요한 사이트 자동화
- 약관상 자동화·크롤링이 금지된 플랫폼 작업
- 모바일 앱(안드로이드/iOS 네이티브) 자동화
- 1일 10,000건 이상 데이터 수집
- 본 SOW에 명시되지 않은 추가 데이터 소스
- 본 SOW에 명시되지 않은 추가 차트·리포트 페이지
- 납품 후 30일 경과 후 발생한 사이트 구조 변경 대응 (월 구독 비가입 시)
- 의뢰자 측 요청에 의한 작업 범위 변경 (Change Order로 별도 처리)
- 데이터 정제 (의뢰자 제공 데이터가 표준 양식과 다른 경우 시간당 8만원 별도)

## 4. 의뢰자(Client) 책임

원활한 작업을 위해 의뢰자는 다음을 제공해야 합니다:

- **API 키 / OAuth 인증**: 착수일 + 3영업일 이내 발급·전달
- **로그인 정보** (필요 시): 착수일 + 3영업일 이내 안전한 채널로 전달
- **샘플 데이터**: 의뢰서 작성 시 또는 착수일 + 1영업일 이내
- **결과물 검토 회신**: 1차 납품 후 5영업일 이내
- **운영 환경 접근권**: 클라우드 계정·서버 접근권 (해당 시)

의뢰자 사정으로 위 항목 제공이 지연되면 작업 일정도 비례하여 연장됩니다. 2주 이상 지연 시 매주 5% 할증.

다음 객관적 기준이 모두 충족되면 인수가 완료됩니다:

- [ ] 본 SOW "2. 포함되는 작업" 모든 항목 동작 확인
- [ ] **5일 연속** 자동 실행 + 결과물 정상 출력 확인
- [ ] 알림 기능(설정된 경우) 정상 발송 확인
- [ ] 운영 가이드 문서 전달 + 화상 인수인계 완료
- [ ] 의뢰자가 본 인수 기준 충족을 확인하는 회신 (또는 회신 5영업일 무응답 시 자동 인수)

## 7. 변경 관리 (Change Order)

작업 진행 중 의뢰자가 본 SOW 범위 외 작업을 요청할 경우:

1. AutoLink SL이 추가 견적·일정 영향을 서면 안내
2. 의뢰자 서면 동의 + 추가 입금 후 진행
3. 추가 동의 없는 작업은 진행하지 않음

추가 비용 없이 처리할 수 있는 사소한 수정(예: 차트 색상 변경, 라벨 수정)은 1차 납품 후 검토 기간 내 1회에 한해 무상.

## 8. 위약·해지 조항

### 8.1. 의뢰자 측 일방 해지
- 착수 후 ~ 1차 납품 전: 진행률에 비례한 정산 + 위약금 (총액의 20%)
- 1차 납품 후 ~ 인수 전: 총액의 70%
- 인수 후: 환불 불가, 잔금 정상 청구

### 8.2. AutoLink SL 측 사유로 진행 불가
- 진행률에 비례한 정산 + 기수령 금액 환불
- 사유: 기술적 불가 판명 / 약관 위반 발견 / 법령 저촉 등

### 8.3. 합의 해지
- 양 당사자 서면 합의 시 진행률 기준 정산
