## 🚦 핵심 원칙 (분쟁 예방)

1. **사전 고지 의무**: SOW 작성 시 **반드시** 외부 API 비용을 명시
2. **무료 대체재 우선**: 무료 옵션이 있으면 1차 제안
3. **사장님 직접 결제 원칙**: 유료 API는 사장님 명의로 가입·결제 (AutoLink SL이 중간 결제 X)
4. **3구간 안내**: 최소/중간/최대 예상 비용 범위 제공
5. **한도 초과 시 통지**: 우리가 대신 결제하는 경우, 사전 합의 한도 초과 시 즉시 통지

- 최소: {x}원 (평균 사용량)

- 최대: {z}원 (이상치)

- 무료 항목: 사장님 부담 없음

  - 가입 절차: {단계}

사장님 동의 없이 추가 비용이 발생하지 않도록 시스템 내 자동 차단 로직을 두겠습니다.

본 운영비 안내에 동의하시면 SOW 회신에 "운영비 안내 확인했습니다" 로 답변 부탁드립니다.

## 🛡 자동 차단 로직 (개발 시 의무 구현)

월 운영비가 사전 합의 한도를 초과할 위험이 있는 경우, 시스템이 **자동으로 작동을 중단**해야 합니다:

### 구현 패턴

```python
MONTHLY_LIMIT_KRW = 30000  # SOW에 명시된 한도
current_month_spend = get_current_month_spend()

if current_month_spend >= MONTHLY_LIMIT_KRW * 0.8:
    notify_owner("⚠️ 월 운영비 80% 도달 — 한도 상향 결정 필요")

if current_month_spend >= MONTHLY_LIMIT_KRW:
    pause_automation()
    notify_owner("🚫 월 운영비 한도 초과 — 자동화 일시 중단됨. 한도 상향 후 재개")
```

매 의뢰 SOW에 **80%·100% 알림 + 자동 중단**을 기본 포함.

## ⚠️ 분쟁 케이스 시뮬레이션

### 시뮬레이션 1: 사장님이 운영비를 모르고 가입

```
[잘못된 케이스]
SOW: "카톡 알림 기본 포함"
사장님: 매월 5만원 이상 청구되는 줄 모름
3개월 후: "왜 매월 5만원씩 빠져나가지?" → 분쟁

[올바른 케이스 — 본 가이드 적용]
SOW: "카카오 알림톡은 사장님 솔라피 계정에서 직접 결제. 예상 월 3,600원~18,000원."
사장님: 사전 인지 + 본인 카드로 직접 결제
3개월 후: 이미 알고 있음, 분쟁 없음
```

### 시뮬레이션 2: AI API 비용 폭주

```
[잘못된 케이스]
SOW에 AI 분석 비용 명시 안 함
의뢰 후 사이트가 갑자기 인기 → AI 호출 10배 → 월 30만원 청구
사장님: "이게 뭐냐?" → 분쟁

[올바른 케이스]
SOW: "AI 분석 월 한도 1만원, 한도 초과 시 자동 중단"
초과 발생 → 자동 중단 + 사장님께 통지 → 한도 상향 또는 사용 줄이기 결정
```
