## 전략 방향
1. "자동화 서비스의 의뢰 접수 자체를 자동화한다." → 의뢰 파이프라인
2. "사이트 자체가 포트폴리오다." → 제작 과정 자체를 기록·홍보 자산화
3. "고객의 업무를 자동화하여 편의를 증진한다"는 메시지를 거대 키네틱 타이포그래피로 강조

---

## 아키텍처 개요

```
[고객] → GitHub Pages (다국어 의뢰서, 실시간 견적)
           ↓ Web3Forms (무료 250건/월)
       Gmail (jyc0289y@gmail.com)
           ↓ Scheduled Task (크론)
       Claude Code 자동 분석
           ├─ 원문 언어 감지 + 한국어 번역
           ├─ 6변수 스코어링 + 매력도 평가
           ├─ 한국어 보고 → PushNotification (차등 알림)
           └─ 응대 초안 3선택지 (적극/표준/거절) → Gmail Draft
                ↓
       진영 대표 선택 → Claude Code가 고객 언어로 번역 → 회신
```

**API 비용 = 0원.** Claude Code(Max 요금제) + Gmail MCP + Scheduled Tasks만 사용.

## 다국적 서비스 확장

### 언어 지원
- 자동 감지: `navigator.language` → IP 기반(ipapi.co) → 영어 폴백
- 우상단 고정 언어 드롭다운, localStorage 저장, 리로드 없이 즉시 전환
- 1차: 한국어, English, 日本語
- 2차: Deutsch, Español, Français, Português

### 의뢰 수신 → 한국어 보고 파이프라인
- 원문 언어 자동 감지 → 한국어 번역 → 스코어링 → 보고
- 응대 초안 A/B/C 한국어 제공 → 선택 → 고객 언어로 번역 회신
- **진영 대표는 한국어로만 의사결정**

### 글로벌 플랫폼 진출
| 단계 | 플랫폼 | 시기 |
|------|--------|------|
| 1차 | 크몽 + 자체 사이트 | Phase 1 |
| 2차 | Fiverr | 후기 3건 후 |
| 3차 | Upwork | 포트폴리오 5건 후 |

### 해외 블라인드 스팟 대응
- 시간대 격차 → 자동 즉시 응답 초안
- 선제적 보고 → 주간 자동 진행 보고 이메일
- SOW 문서 → 영문 견적서 포함
- GDPR → 차단 문구에 EU 별도 협의 추가
- 결제 → 플랫폼 결제 우선, Wise 계정 준비
