## 🔴 ACT-007: Gmail MCP 복구 (우선순위 최상위)

### 왜 critical한가?
- 모든 의뢰가 Gmail로 도착하고, 응대 초안도 Gmail Draft로 생성됨
- 현재 Chrome MCP로 우회 중이지만 5~10배 느리고, 노트북이 항상 켜져 있어야 함
- Phase 2 (Fiverr 진출) 진입 전 반드시 복구 → 해외 고객 응답 지연 24h = 이탈률 30%
- **응답 누락 1건 ≈ 420만원 LTV 손실** (시그니처 #4 Premium 기준)

### 절차 (5~10분)

#### Step 1: Google 권한 캐시 정리

브라우저에서:
```
1. https://myaccount.google.com/permissions 접속
   (계정: jyc0289y@gmail.com 로 로그인되어 있는지 확인)

2. "타사 앱 또는 서비스" 또는 "Third-party apps with access" 섹션

3. "Anthropic" 또는 "Claude" 또는 "Claude AI" 항목 찾기
   → 항목 클릭 → "액세스 권한 삭제" / "Remove Access" 클릭
   → 확인
```

#### Step 2: Claude 데스크탑에서 Gmail 재연결

```
1. Claude 데스크탑 앱 → Settings (⌘+,) → Connectors

2. 기존 Gmail 항목이 있으면 "제거" 클릭

3. "+ 추가" 또는 "Add Connector" → "Gmail" 선택

4. Google 로그인 화면 → jyc0289y@gmail.com 선택

5. 권한 화면에서 다음 모두 체크:
   ✅ Gmail 메시지 보기 (gmail.readonly)
   ✅ Gmail 라벨 관리 (gmail.labels)  ← 핵심
   ✅ Gmail 초안 작성 (gmail.compose) ← 핵심
   ✅ Gmail 메일 수정 (gmail.modify)

6. "Allow" / "허용" 클릭
```

#### Step 3: Claude에 알리기

복구 완료 후 Claude 채팅에 다음 메시지:
```
Gmail MCP 복구 완료. 라벨 5개 자동 생성 + SKILL.md API 버전 전환 진행해줘.
```

→ Claude가 즉시:
- 5개 라벨 자동 생성 (`AutoLink/신규`, `검토중`, `수주`, `거절`, `처리완료`)
- `SKILL.md`를 API 기반으로 재전환
- ACT-008 자동 처리 (라벨 수동 생성 불필요해짐)
- REMINDERS.md에서 ACT-007 ✅ 처리

### 만약 Step 2에서 Google이 400 에러를 낸다면

대안 절차:
```
1. 시크릿 모드/InPrivate 창 열기
2. claude.ai 또는 데스크탑 앱 사용 (가능하면 둘 다 시도)
3. Google 권한 화면이 나오지 않으면:
   → https://myaccount.google.com/permissions 한 번 더 확인
   → Anthropic 항목이 정말 삭제됐는지 재확인
   → 5분 대기 후 재시도
```

여전히 안 되면 Claude에 "Gmail MCP 400 에러 계속됨"이라고 알려주시면 다른 우회 방안 검토.

## 🟡 ACT-008: Gmail 라벨 5개 수동 생성

### 왜 필요한가?
- ACT-007 (Gmail MCP 복구)가 완료되면 자동 생성됨
- 그 전까지는 Chrome MCP 우회 SKILL이 작동하려면 라벨이 미리 만들어져 있어야 함

### 절차 (3분)

```
1. https://mail.google.com/mail/u/jyc0289y@gmail.com/ 접속

2. 좌측 사이드바 하단 "더보기" → "+" (라벨 추가)

3. 다음 5개를 순서대로 생성:
   ① AutoLink/신규
   ② AutoLink/검토중
   ③ AutoLink/수주
   ④ AutoLink/거절
   ⑤ AutoLink/처리완료

   ※ 슬래시(/)를 사용하면 Gmail이 자동으로 "AutoLink" 부모 라벨 아래
     중첩 라벨로 그룹화합니다.
```

### 검증

생성 후 좌측 사이드바에서 다음 구조 확인:
```
📁 AutoLink
   ├─ 신규
   ├─ 검토중
   ├─ 수주
   ├─ 거절
   └─ 처리완료
```

ACT-007 복구 시 라벨이 자동 생성되므로 이 작업은 **임시 우회 운영 기간 동안만 필요**.

## 🟡 ACT-002: autolinksl.com 도메인 구매

### 추천 옵션

| 등록 업체 | 첫 해 | 갱신 | 추천 사유 |
|----------|-------|------|----------|
| **Cloudflare Registrar** | $9.15 | $9.15 동일 | 가성비 최고, 갱신 가격 변동 없음 |
| 가비아 (한국) | ₩16,000 | ₩24,000 | 한국 결제 편리, UI 친숙 |
| Cafe24 | ₩22,000 | ₩22,000 | 첫해 할인 이벤트 종종 있음 |
| Porkbun | $5~7 (첫해) | $10~11 | 매우 저렴, 영어 결제 |

→ **추천: Cloudflare Registrar** ($9.15/년 영구 동일가, 갱신 폭탄 없음)

### Cloudflare Registrar 절차 (15분)

```
1. https://www.cloudflare.com 가입 (무료)

2. Registrar → "Register a domain" → "autolinksl.com" 검색
   (또는 autolink-sl.com, sl-autolink.com 도 검토)

3. 사용 가능 확인 → 카드 등록 → 결제

4. DNS 자동 설정 (Cloudflare가 nameserver 자동 할당)

5. GitHub Pages 연결:
   - Cloudflare DNS 패널 → "+ 추가"
   - Type: CNAME, Name: @, Target: jyc0289y-art.github.io
   - Type: CNAME, Name: www, Target: jyc0289y-art.github.io
   - Proxy: 켜기 (주황색 구름) — Cloudflare CDN/HTTPS 자동

6. GitHub Pages 설정:
   - https://github.com/jyc0289y-art/autolink/settings/pages
   - "Custom domain" 칸에 autolinksl.com 입력 → Save
   - "Enforce HTTPS" 체크 (DNS 전파 후 자동 활성화)
```

### 검증

10~30분 DNS 전파 후 `https://autolinksl.com`에서 사이트 로드 확인.

도메인 구매 후 Claude에 "autolinksl.com 구매 완료, GitHub Pages 연결도 끝났어"라고 알리면:
- README, _INDEX.md 등 문서의 URL을 새 도메인으로 갱신
- 크몽 시안에 새 도메인 반영
- Web3Forms 폼 도메인 화이트리스트에 추가 (Web3Forms 앱 내 설정)

## 🟡 ACT-006: 변호사·변리사 상담

### 왜 필요한가?
- 현재 [`legal/templates/terms-of-service.md`](../legal/templates/terms-of-service.md)는 **자체 작성본** (변호사 검토 안 됨)
- B등급(100만원+) 수주 시 분쟁 발생 시 약관 효력이 약할 수 있음
- AutoLink SL 상표 출원도 변리사 도움이 필요

### 변호사 상담 (약관 검토)

**시점**: 첫 B등급(100만원 이상) 의뢰가 들어와 수주가 가시화됐을 때

**비용**: 자영업자 대상 표준 거래 약관 검토 50~100만원

**상담처**:
- 대한변호사협회 변호사 검색: https://www.koreanbar.or.kr (지역+분야: IT 계약/약관)
- 로톡: https://www.lawtalk.co.kr (IT 계약 전문 검색)
- 무료 1차 상담: 서울시 무료법률상담 (https://help.seoul.go.kr) — 7~10분 간단 자문

**중점 검토 항목**:
1. 책임 한도 (현재: 거래 총액) — 적정 수준인지
2. 면책 조항 (사이트 측 정책 변경) — 한국 법원에서 인정되는 표현인지
3. 분쟁 관할 (서울중앙지법) — 해외 고객은 별도
4. 위약금 (20%) — 약관규제법상 적정성
5. 개인정보 처리 (5년 보관) — 전자상거래법 준수 여부

### 변리사 상담 (상표 출원)

**시점**: 브랜드 확정 + 도메인 구매 후 (즉, ACT-002 완료 후)

**비용**:
- 출원료: 약 20만원
- 변리사 수임료: 50~150만원 (1류 기준)
- 등록료: 등록 결정 후 추가 약 22만원
- **합계 약 90~200만원** (1~2년 소요)

**상담처**:
- 대한변리사회: https://www.kpaa.or.kr
- 비대면 상담: PatentBomb, 마크커리어 등 온라인 출원 대행 서비스 (저가 30~50만원, 단순 케이스 적합)

**우리 상황 (BMW 35류 등록 vs AutoLink SL)**:
- 변리사가 **유사 판단 가능성**을 평가해야 함
- 출원 분류: 9류 (소프트웨어), 42류 (SaaS·데이터 처리) 핵심
- **35류는 BMW가 점유 중**이므로 35류는 출원하지 않거나 SL 식별력을 강조하는 디자인 출원 필요

**우선순위**:
1. 9류 (S/W) + 42류 (SaaS) 우선 출원 — 핵심 영업 분류
2. 35류는 변리사 권고에 따름 (BMW 우선권 충돌 회피)
