# 크몽 상세페이지 시안 (AutoLink SL)

## 제목 (60자 이내)
**[자동화+대시보드] 반복 업무 끝, 웹사이트로 실시간 확인 + AI 분석 포함**

## 카테고리
- 1차: IT·프로그래밍 > 업무 자동화
- 2차: IT·프로그래밍 > 데이터 수집·크롤링

## 썸네일 컨셉
- 흰 배경, 굵은 검정 타이포 "반복 업무, 자동으로."
- 우측 하단 작은 대시보드 미니어처
- AutoLink SL 로고 (좌상단)

---

## 패키지 (3단 구성)

### Basic — 50만원
**[일회성] 자동화 스크립트만 필요한 분**
- Playwright 기반 크롤링 스크립트 1종
- CSV/엑셀 결과물 출력
- 실행 가이드 문서 포함
- 1회 납품 (유지보수 별도)

### Standard — 150만원 ⭐ 인기
**[자동화 + 웹 대시보드] 사장님이 손 안 대고 보고 싶으신 분**
- Basic 전체 +
- 웹 대시보드 (PC/모바일 반응형)
- GitHub Actions cron 자동 실행
- 차트·테이블 시각화
- 월 1회 동작 점검 (월 5만원 구독 시)

### Premium — 350만원
**[실시간 + 알림 + 다중 사용자] 본격적인 운영용**
- Standard 전체 +
- 실시간 데이터 갱신
- 카톡/이메일 알림 (변동·이상 징후)
- 다중 사용자 로그인
- AI 분석 리포트 (Claude 기반)
- 월 10만원 구독 (호스팅·모니터링·셀렉터 변경 대응)

## 차별화 4줄 (굵게)

1. **엑셀이 아닌 사이트로 받습니다** — 다른 외주자 99%는 엑셀만 줍니다
2. **AI 분석 리포트 포함** — Claude가 데이터를 자동 해석·요약
3. **카톡 알림 기본 포함** — PC 안 켜도 됩니다
4. **월 5만원에 대신 운영** — 사장님은 손 안 댑니다

## 시그니처 케이스 (포트폴리오)

### 케이스 #1: 가격 모니터링 (예: 스마트스토어 셀러)
> "경쟁사 10곳 가격을 매일 자동 수집. 가격 변동 시 카톡으로 알림. 대시보드에서 한눈에 비교."
- 권장 패키지: Standard
- 견적: 150만 + 월 5만

### 케이스 #2: 음식점 리뷰 통합 + AI 감성 분석
> "네이버지도·카카오맵·배민 리뷰를 매일 수집. AI가 긍정/부정 자동 분류. 주간 리포트 자동 생성."
- 권장 패키지: Premium
- 견적: 300만 + 월 10만

### 케이스 #3: 이커머스 매출 통합 대시보드
> "네이버스마트스토어·쿠팡·자사몰 매출을 하나의 대시보드에서 실시간 확인. 매장별·상품별 KPI."
- 권장 패키지: Premium
- 견적: 500만 + 월 20만

## 작업 프로세스 (5단계)

1. **사전 질문지 작성** (5분) — 자동화 대상 사이트, 결과 형식, 실행 빈도 등 6개 항목
2. **샘플 페이지/파일 1건 공유** — 정확한 견적 산출을 위한 필수 단계
3. **15분 화상 통화** (B등급 이상) — 요구사항 상세 확인
4. **견적서 PDF + 작업 범위 명세서** — 포함/미포함 명시 후 서명
5. **계약금 50% 선입금 후 착수** — 크몽 안전결제 활용

## 서비스 범위 (필수 안내)

### 포함되는 작업
- 약관상 자동화가 허용된 공개 웹사이트 크롤링
- 공개 API 연동 (네이버 검색 API, 카카오맵 API 등)
- 엑셀·CSV·구글시트 데이터 처리
- 정적/동적 웹 대시보드 제작
- GitHub Actions cron 자동화

### 포함되지 않는 작업 (별도 견적)
- 의뢰자 회사 내부 시스템(ERP·그룹웨어·POS) 연동
- 캡차·SMS 인증·생체 인증이 있는 사이트
- 약관상 자동화가 금지된 플랫폼
- 모바일 앱 자동화 (안드로이드/iOS 네이티브)
- 1일 10,000건 이상 데이터 수집
- 납품 후 30일 경과 후 발생한 사이트 구조 변경 대응

## FAQ

**Q1. 비용은 어떻게 정해지나요?**
A. 6가지 항목(데이터 소스 수·종류·가공 로직·결과물 형태·실행 빈도·데이터 품질)으로 복잡도를 점수화하여 등급별 단가를 안내합니다. **프로필 웹사이트의 의뢰서를 작성하시면 즉석 견적이 산출됩니다.**

**Q2. 납품 후 사이트가 바뀌면 어떻게 되나요?**
A. 월 구독 가입 시 셀렉터 변경 대응이 무료 포함됩니다. 비구독 시 회당 10만원 추가 비용이 발생합니다.

**Q3. 어디까지 자동화할 수 있나요?**
A. 약관상 자동화가 허용되고 캡차가 없는 공개 웹사이트는 대부분 가능합니다. 공개 API가 있으면 우선 활용하여 안정성을 높입니다.

**Q4. 해외 사이트도 가능한가요?**
A. 네. 영어·일본어 등 어떤 언어의 사이트든 가능합니다. 또한 해외 고객도 환영합니다 — 의뢰서를 어떤 언어로 작성하셔도 응대해드립니다.

**Q5. 결과물 데모를 볼 수 있나요?**
A. 네. **프로필의 웹사이트에서 시그니처 케이스 데모와 실시간 견적 계산기를 확인하실 수 있습니다.**

## 외부 링크 안내 (크몽 정책 준수)

크몽 정책상 상세페이지 본문에 외부 URL 직접 삽입이 불가하므로, 아래와 같이 안내:

> "더 자세한 포트폴리오와 의뢰서 양식은 **크몽 프로필의 웹사이트 필드**에서 확인하실 수 있습니다."

크몽 1:1 메시지에서는 URL 전달 가능 — 첫 문의 시 자체 사이트 URL 안내.
